## 在search中进行搜索（最好的方案）

- 在url上进行搜索，在?前面 一个单词接着一个单词搜

​            https://music.163.com/weapi/song/enhance/player/url/v1?csrf_token=

​            1. v1

​            2. url/v1

​            3. player/url/v1


​            或者该url中某些看着比较特殊的单词. 也可以单独搜索 ...

- 搜参数(请求上的各种参数(主要针对加密参数)

- 拦截器. 该网站使用的是异步. promise
